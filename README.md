# java-web-sbt-blade-postgres-hibernate-simple

## Description
Creates a small database table
called `dog` and populates with
hql.

## Tech stack
- java
- sbt
  - hibernate
  - hql
  - blade
  - postgres driver

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10
- postgres:alpine

## To run
`sudo ./install.sh -u`
[Available here](localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
